# 刻纸底稿转换工具

上传一张图片，转成能直接刻的黑白底稿，阴刻、阳刻两版一起出。内置 OpenCV 图像内核（WASM），复杂彩图、插画、照片都能出干净、认得出的线稿。纯本地运行，图片不上传服务器。

**约定：黑色 = 保留的纸，白色 = 刻掉镂空的部分。**

## 在线使用（手机 / 电脑浏览器直接打开）

👉 **https://erikaalk.github.io/kezhi-web/**

把这个网址发给别人就能用，无需安装。首次打开要加载约 10MB 的算法内核，稍等几秒；特别老的手机（如很旧的 iPhone）可能加载慢或吃力。

## 文件说明（下载到本地用时，两个文件要放在一起）

- `index.html` —— 主程序，双击用浏览器打开。
- `opencv.js` —— 图像算法内核（约 10MB），**必须和 HTML 放在同一个文件夹**，否则打不开。

首次打开会加载约 10MB 的内核，稍等两三秒；之后就快了。全程离线，不联网。

## 怎么用

1. 把这两个文件放同一文件夹，双击 `刻纸底稿转换工具.html`。
2. 等左上角提示「加载内核」结束后，把图片拖进去，或点上传、或 `Ctrl+V` 粘贴。
3. 左边选风格、调参数，右边实时出「阳刻底稿」和「阴刻底稿」两版。
4. 点「下载 PNG」保存，打印出来就能照着刻；也能存 SVG（给激光切割用）。

## 三种风格

- **线描**（默认）：用 OpenCV 边缘检测勾出线稿，线线相连成整体。复杂插画、照片首选，出来干净、认得出。
- **块面**：按明暗分成大块保留，厚重。适合想要「面」的效果、剪影感强的图。
- **剪影**：整体一个阈值，适合本身对比就强、主体分明的图（纯色 logo、剪纸图样）。

## 阴刻 vs 阳刻

- **阳刻（阳文）**：留下来的是线 / 主体（黑），四周大量镂空，以线造型、线线相连。
- **阴刻（阴文）**：留下来的是底子（黑），线条 / 主体镂空成白，以面造型、块块相连。基本是阳刻的反色。

两版每次都一起给，挑顺眼的刻就行。

## 参数

- **取线门槛**（线描）：越高线越少、越干净；越低线越多、越细。
- **降噪平滑**（线描）：压掉毛发、纹理这类杂线。
- **明暗平衡 / 保边强度 / 内部细节线沟**（块面）：控制黑白比例、抹平噪点、在黑块里刻白线表现细节。
- **反相**：主体偏亮、或出来黑白反了，勾上它。
- **线条 / 边缘加粗**：让线更粗、更结实、不易刻断。
- **去除杂点**：清掉细碎的小块。
- **自动加连接筋**：核心功能，见下。

## 可刻性：连接筋

刻纸是一整张纸，镂空之后剩下的纸必须连成一体，否则会掉块散架。工具会自动找出「孤岛」——那些没跟主体连着的保留块，用最短的直线给它加一条细「连接筋」拉回主体，也就是传统剪纸里的「留筋」；离得太远又太小的碎点则直接清掉。

每一版右上角会标「已连通 / 未连通」。若还没完全连通，调大「连接筋 / 加粗」，或调低「去除杂点」。

## 技术说明

- 前端 + OpenCV.js（WebAssembly），无需服务器、无需安装、离线可用；图片不出本机。
- 线描管线：灰度 → 双边保边滤波 → Canny 边缘检测 → 形态学闭合 → 连通域去杂点 → 多源 BFS 加连接筋。
- 块面管线：双边保边滤波 → Otsu 色调分块 →（可选）Canny 线沟 → 形态学清理 → 连接筋。
- 处理尺寸默认最长边 850px，可在「输出」里调。

## 更新记录

### v2.0（2026-07-12）

- 换用 OpenCV.js 图像内核（内嵌本地 WASM，离线可用），复杂彩图/插画的线稿质量大幅提升，不再糊成一坨。
- 线描改为 Canny 边缘检测（连成整体、认得出主体），设为默认风格；块面、剪影保留为可选。
- 连接筋改为「直线连最近落点 + 远小碎点直接清除」，避免孤岛过多时连出乱线。

### v1.0（2026-07-12）

- 首版：纯 JS 实现，块面 / 线描 / 剪影三风格，阴阳双版输出，自动连接筋，PNG / SVG 导出。
